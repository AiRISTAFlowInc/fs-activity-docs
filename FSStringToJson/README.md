# StringToJson

This activity parses a string to a json object.

## Configuration

### Input:
| Name       | Type   | Description
|:---        | :---   | :---    
| StringData | string | The String data to be converted

### Output:

| Name   | Type    | Description                       |
| :----- | :------ | :-------------------------------- |
| JsonObject | object | The Json object
